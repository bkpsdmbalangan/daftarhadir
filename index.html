Prosedur Reset Akun (Lupa Password)

Jika pengusul lupa password, Verifikator dapat membantu dengan langkah berikut:

  1. Buka Firebase Console > Menu Authentication.
  2. Cari email: [NIP]@simpelblg.id.
  3. Klik titik tiga (⋮) > Delete Account.
  4. Buka menu Realtime Database > Node pegawai/[NIP].
  5. Hapus baris status_gelarku dan email_aktif.
  6. Informasikan pegawai untuk melakukan Aktivasi Akun ulang.

Peringatan Keamanan Kredensial

Sistem TIDAK menyediakan fitur "Lupa Password". Saat melakukan Pembuatan Akun, WAJIB untuk mencatat atau men-screenshot Password Anda.

1

Aktivasi & Login

  • Klik tombol "Daftar di sini!" di layar awal.
  • Masukkan 18 digit NIP tanpa spasi, klik "Cek", lalu buat password yang terdiri dari gabungan huruf dan angka minimal 6 karakter.
  • Gunakan NIP dan Password tersebut untuk MASUK (Login) ke sistem.
2

Tambah Usul & Unggah Berkas

  • Di halaman dasbor, klik tombol "Tambah Usul".
  • Isi seluruh kolom formulir usulan dengan lengkap.
  • Sistem akan membuatkan Folder Drive khusus. Klik tombol "Unggah Berkas" untuk mengunggah berkas (PDF) ke dalam folder tersebut.
3

Pantau Status & Edit Data

  • Status berkas Anda akan terlihat di Dasbor.
  • Jika status BTS (Perlu Perbaikan), baca dan tindak lanjuti catatan dari Pengelola/Verifikator.
  • Gunakan tombol "Cek & Edit Data" jika ada kesalahan input pada formulir.
  • Gunakan tombol "Beri Tanggapan" untuk mengabari petugas bahwa Anda telah memperbaiki dokumen.
4

SK Terbit & Usul Baru

  • Jika SK telah diterbitkan, status akan berubah menjadi Terbit SK.
  • Tombol baru "Lihat SK" akan muncul untuk mengunduh file SK tersebut.
  • Jika di tahun-tahun berikutnya Anda ingin mengusulkan pencantuman gelar / pendidikan lain, cukup klik tombol "Tambah Usul" lagi.

Tugas Pengelola Unor Induk

  • Pantau Usulan: Cek tab "Rekap Usul" secara berkala untuk melihat usulan baru dari pegawai di SKPD Anda.
  • Periksa Berkas: Klik tombol Berkas untuk membuka folder Google Drive dan memverifikasi kelengkapan berkas.
  • Berikan Keputusan: Klik tombol Verifikasi untuk mengubah status menjadi ACC Unor, BTS, atau TMS dan berikan catatan yang relevan. Anda juga bisa mengedit data jika terdapat kesalahan pengetikan dari pengusul.

Tugas Verifikator Instansi

  • Buka/Tutup Pengusulan: Gunakan tab "Pengaturan" untuk membuka atau menutup akses pengusulan aplikasi bagi seluruh ASN.
  • Verifikasi Final: Berikan status akhir (ACC Instansi / ACC BKN). Gunakan kolom Keterangan Internal untuk saling berkoordinasi sesama Verifikator.
  • Unggah SK: Jika SK telah terbit, ubah status menjadi Terbit SK dan lampirkan (upload) file PDF-nya pada kolom yang disediakan.
  • Reset Akun: Jika ada pegawai yang lupa password, gunakan fasilitas "Prosedur Reset Akun" di Tab Pengaturan.
Fitur "Unduh Rekap" ke Excel tersedia untuk memudahkan pelaporan administrasi bulanan/tahunan.
